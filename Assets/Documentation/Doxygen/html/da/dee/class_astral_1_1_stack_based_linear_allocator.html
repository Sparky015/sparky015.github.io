Detailed Description

template<size_t MemoryBlockSize>
class Astral::StackBasedLinearAllocator< MemoryBlockSize >

A stack-based linear allocator. Max allocation size is 5.28 KB due to being on the stack. Deallocate method does nothing. Reset method deallocates the whole memory block.

Warning
You have to use the Reset method to Deallocate memory. It deallocates all memory being used. It's all or nothing.
Thread Safety
This class is NOT thread safe.
