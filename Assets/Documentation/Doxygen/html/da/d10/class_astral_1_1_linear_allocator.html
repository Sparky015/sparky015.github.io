Detailed Description

A heap-based linear allocator. Deallocate method does nothing. Reset method deallocates the whole memory block.

Warning
You have to use the Reset method to Deallocate memory. It deallocates all memory being used. It's all or nothing.
Thread Safety
This class is NOT thread safe.
