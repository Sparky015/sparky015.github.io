Moves a specified piece from its current location to the specified location.

Parameters
colorThe color of the piece being moved.
pieceIndexThe index of the piece being moved.
targetBoardLocationThe location that the piece is being moved to. Expected range is 0-63 inclusive.
Exceptions
std::logic_errorErrors when the square being moved to is already occupied.
std::out_of_rangeErrors when the specified location is not 0-63 inclusive.
Note
The move piece functionality is separated from the take piece functionality, so it will throw errors when the intention is to move to a square that is empty, but the square actually isn't. This assumes that there is a valid check being made to see if the move is possible prior to this function's call.
