◆ MemoryUnitLabelHelper()

std::string_view Debug::MemoryUnitLabelHelper ( size_t numberOfBytes)

Dynamically adjusts the unit label of the memory used based on the number of bytes being represented.

Parameters
numberOfBytesThe number of bytes that are being represented
Returns
A string representation of the memory size with the corresponding unit label
Note
The unit label only goes up to TB
Warning
The returned string_view is valid only until the next call to this function from the same thread. Do not store or cache the returned string_view.
