Detailed Description

template<typename T>
class Core::STLStackAllocator< T >

An STL compliant version of the StackAllocator. It is a stack-like allocator that allocates memory in a last in first out order. This means that the user can deallocate only the most recent unfreed memory allocation.

Thread Safety
This class is NOT thread safe.
