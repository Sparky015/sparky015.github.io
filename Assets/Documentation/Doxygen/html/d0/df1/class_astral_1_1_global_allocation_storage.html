void Astral::GlobalAllocationStorage::AddPointer ( const AllocationData & allocationData)

Stores a mapping of the pointer to the allocation size of the pointer.

Remarks
Does nothing when pointer is nullptr.
Warning
This method will do nothing if the pointer is already being stored. You can not update pointers with new information until it is freed from storage.
