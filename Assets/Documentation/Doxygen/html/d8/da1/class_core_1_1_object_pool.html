Detailed Description

template<typename ElementType, size_t NumberOfElements, typename ElementAllocator = std::allocator<ElementType>>
class Core::ObjectPool< ElementType, NumberOfElements, ElementAllocator >

A generic object pool. Constructs many objects at once and lends them out. Objects are not reconstructed and will maintain the state when released back to the pool.

Thread Safety
This class is not thread safe. User must handle thread safety if needed.
