bool Astral::AllocatorUtils::ResizeMemoryBlock ( void * memoryBlockPtr,
size_t memoryBlockSize,
void *& outNewMemoryBufferPointer,
size_t & newMemoryBufferSize )
nodiscard

Resizes an allocator's memory block to a bigger size. This frees the old pointer only if resize allocation succeeds.

Parameters
memoryBlockPtrPointer to the allocator's memory block.
memoryBlockSizeThe current size of the allocator's memory block
outNewMemoryBufferPointerA pointer to the new allocated block that is resized by the given. Will be nullptr if resize allocation failed.
newMemoryBufferSizeThe amount the original memory block size is multiplied by to get the new memory block's size.
Returns
True if the resize operation succeeded and false if the resize allocation failed
Warning
This does not copy any data over. It is intended to be used when the allocator is empty.
Thread Safety
This function is not thread safe.
